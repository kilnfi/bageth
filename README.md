# Bageth

Ethereum Explorer for the Beaconchain

<img src="./static/header.jpg" />

Currently includes:

- Stake explorer

## Developing

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

## Building

To create a production version of your app:

```bash
pnpm build

# preview with
pnpm preview
```
